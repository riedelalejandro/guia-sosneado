## Notas para el Escalador en la Montaña

### Sistema de graduación
El blog usa el **sistema francés** (5+, 6a, 6a+, 6b, 6b+, 6c, 6c+, 7a, 7a+, 7b, 7b+).

### Equipamiento general recomendado
- **Siempre:** 2 juegos completos de camalots (#0.3 al #4)
- **Para offwidth:** camalot #5 y posiblemente #6
- **Empotradores/stoppers** variados para las agujas con fisuras finas
- **2 cuerdas de 60m** para las rutas largas (Principal, Ectelion, Misil)
- **1 cuerda de 50m** suficiente para agujas cortas (Bolillo, S2, Cóndor)
- Material de reunión: cordinos, mosquetones HMS

### Descenso general
La mayoría de las agujas tienen rappeles equipados con chapas con argolla. Para los más naturales, llevar **cintas y cordinos para abandonar**.

### Señales de peligro ⚠️
- Algunos largos tienen advertencias de **roca descompuesta** (especialmente cara sur de la Principal, partes de Ectelion/Chaparral)
- La **cara sur de la Principal** solo recomendada en verano o con mucho viento
- Varios largos en agujas remotas (El Rey, Fortaleza) sin reunión fija — **fundamental orientación y experiencia**
